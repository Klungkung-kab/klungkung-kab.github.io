# Portal Resmi Kabupaten Klungkung - GitHub Pages

Website portal informasi untuk **Kabupaten Klungkung**, Bali. Menyediakan informasi umum, profil daerah, dan blog artikel.

## 🌐 Live Website

- **Portal Utama**: [https://klungkung-kab.github.io](https://klungkung-kab.github.io)
- **Halaman Blog**: [https://klungkung-kab.github.io/blog](https://klungkung-kab.github.io/blog)

## 📋 Fitur

- ✅ **SEO Friendly** - Meta tags lengkap, semantic HTML, structured data
- ✅ **Responsive Design** - Tampilan sempurna di semua perangkat
- ✅ **Informasi Lengkap** - Profil daerah, visi misi, data geografis
- ✅ **Blog Otomatis** - Daftar artikel di `/blog` ter-update otomatis via GitHub API
- ✅ **Aksesibilitas** - ARIA labels dan keyboard navigation
