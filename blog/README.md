
## 🚀 Cara Menambahkan Artikel Blog

1. Buat file `.html` baru di folder `/blog/`
2. Upload ke repository (commit & push)
3. Halaman [https://klungkung-kab.github.io/blog](https://klungkung-kab.github.io/blog) akan **otomatis** menampilkan artikel baru

> **Tips SEO**: Gunakan nama file yang deskriptif (contoh: `profil-klungkung.html`, `wisata-nusa-penida.html`)

## 🛠️ Teknologi

- HTML5 Semantic
- Tailwind CSS (CDN)
- Font Awesome Icons
- JSON-LD Structured Data
- GitHub API (untuk auto-list blog)
